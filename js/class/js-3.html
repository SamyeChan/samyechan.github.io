# if 语句语法
 
  1. 最简单的，只有“如果”：
 
if (判断条件) {

  1. 中级，“如果...否则”：
 
if (判断条件) {
  // 满足条件执行
} else {
  // 否则情况下执行
}

  1. 高级，“如果...或者如果...或者...否则”
 
if (判断条件1) {
  // 满足条件1执行
} else if (判断条件1) {
  // 满足条件2执行
} else {
  // 否则情况下执行
}

# 注意事项
 
  • 判断条件可以有多个；
    •  
  • 可以只有 if 没有 else；
 
# 比较运算符和逻辑运算符
 
  • A == B - 比较等号左右两边的值是否相等，相等返回 true，否则返回 false；
    •  
  • A && B - 若左侧结果为 true，则返回右侧结果，否则，直接返回 false；
    •  
  • A || B - 若左侧结果为 true，则直接返回左侧结果而不会对右侧进行计算，否则返回右侧结果；
 
# 逻辑运算符
 
  1. 与 - &&；
    2.  
  2. 或 - ||；
    3.  
  3. 非 - ! → 取反；
 
# 判断条件之布尔值
 
  • true - 真；
    •  
  • false - 假；
 
判断条件的小括号：程序会将括号内的判断条件转为 true 或 false，而后交给 if 做判断；
 
# a 标签是超链接，会导致页面的刷新或者页面跳转，所以我们可在 href 添加 javascript:; 来禁止这些情况的发生；
 
# 数组
 
  • [] - 可以存储多个值；
    •  
  • 每个值之间利用逗号进行间隔，最后一个值后没有逗号；
 
# 数组的长度
 
  • arr.length - 查看当前数组值个数；
 
# 数组的取值
 
  • 根据下标进行取值；
    •  
  • 下标为数字，并不符合标识符命名要求 → 利用 [] 进行操作，而非 .；
    •  
  • 下标从 0 开始；
 
# for 循环
 
  • 有多个循环变量的情况下，在最后一个条件不成立时跳出（而非最小/最大化情况下）；
    •  
  • 循环是在js代码读取到这儿时就执行了；
    •  
  • 事件函数中不要调用外部循环变量，因为...就相信我啊，循环早就执行完了好吧...

# classList
 
el.classList 获取设置元素的 class 列表 → 类数组
 
  • 通过下标获取元素的每一个 class；

    •  
  • add('class名') - 给元素 classList 添加一个 class；
    •  
  • remove('class名') - 从元素的 classList 中删除一项；

# 网络 | HTTP

# HTTP 是什么

  • = HyperText Transfer Protocol，超文本传输协议；
  • 一种可扩展的网络传输协议 → 应用层协议；
  • 基于 TCP/IP 通信协议传递数据；
  • 在 Web 上进行数据交换的基础，是 client-server 协议；
  • 所有 WWW 文件都必须遵守这个标准；
  • 客户端发出的消息为 requests，被服务器响应的消息为 responses；

# HTTP 组成 & 特点

  • HTTP = 请求报文 + 响应报文；
  • HTTP请求 = 请求行 + 请求头部 + 空行 + 请求数据；
  • HTTP响应 = 状态行 + 消息报头 + 空行 + 响应正文；
  • 特点：简单快速 / 灵活 / 无连接 / 无状态；

# HTTP 请求方式

8种

  1. OPTIONS
  2. HEAD
  3. GET
  4. POST
  5. PUT
  6. DELETE
  7. TRACE
  8. CONNECT

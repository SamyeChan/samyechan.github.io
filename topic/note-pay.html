# 你需要知道的“支付”

阅读 |《做支付需要了解哪些行业知识》

# 二清

  • 即 二次清算；
  • 支付机构 → 平台 → 商户/客户； 支付机构将资金结算给平台，平台再自行将资金清算给下游的商户/客户
  • 若平台无清算资质（无支付牌照），即违规二清；
  • 需防范平台吸收资金后挪用其进行高风险投资，甚至跑路；

# 支付牌照

  • 即 《第三方支付的业务许可证》，央行颁发，并受其监管；
  • 三种牌照：银行卡收单（POS机）、互联网支付、预付卡发行与受理；

  • 支付牌照之所以抢手，是因为其申请门槛极高，需央行审核，并且，央行几乎已停止发牌照了；
  • 目前全国仅存200家左右的第三方支付公司，而可做互联网支付的或许不到100家了；
